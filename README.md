🚀 Interview Simulator

🔗 Live Demo: https://interview-simulator-lake.vercel.app/

An AI-powered Interview Simulator that replicates real interview scenarios to help users practice, improve confidence, and prepare effectively for technical and non-technical roles.

📌 About the Project

The Interview Simulator is built to solve a common problem:

Lack of real interview practice before actual job interviews.

This platform allows users to simulate interviews based on their role, skills, and experience level, providing a structured and realistic preparation environment.

✨ Key Features
🎯 Role-Based Interview Generation
Generate interview questions tailored to specific job roles
🤖 AI-Powered Question Engine
Dynamic and intelligent question generation
🧠 Skill-Based Customization
Questions adapt based on selected skills & experience
🎤 Real Interview Simulation Flow
Step-by-step interview experience
⚡ Fast & Responsive UI
Smooth experience across all devices
